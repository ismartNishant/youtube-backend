# YouTube-Style Backend

This project is a backend service inspired by YouTube. It handles video data, user channels, comments, likes, and basic streaming APIs.

## Features

- Video upload and metadata management
- Channel and user profile support
- Comments and likes
- Search and list videos
- API routes for frontend integration

## Getting Started

1. Install dependencies
2. Configure database and environment variables
3. Run the server

## Notes

This backend is designed to serve a video platform with features similar to YouTube.
